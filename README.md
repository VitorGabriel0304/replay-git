# Relembrando comandos Git/Github

## Configurações 
adiciona o nome do desenvolvedor´
```bash
git config --global user.name "Vitor Gabriel Vido"
```
adiciona o e-mail do desenvolvedor
```bash
git config --global user.email vitorlatero03@gmail.com
```
## Comandos GIT
Roda apenas uma vez para executar o git dentro da pasta
```bash
git init
```

Cria uma copia dos arquivos para subir em nuvem
```bash
git add .
```
Verifica a atual situação dos arquivos
```bash
git status
```

Sobe o comentario sobre a atualizção dos arquivos
```bash
git commit -m "comentario"
```

Sobe os arquivos para o repositorio 
```bash
git push
```
